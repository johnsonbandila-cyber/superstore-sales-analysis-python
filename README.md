# 📊 Superstore Sales Analysis using Python

## 📌 Project Overview

This project analyzes Superstore sales data using **Python**, **Pandas**, and **Matplotlib**. The dataset was divided into four separate tables (Sales, Orders, Products, and Shipping) to simulate a real-world relational data model.

The project focuses on data exploration, data modeling using Pandas, business insights, and data visualization.

---

## 🎯 Objectives

- Analyze sales performance across different categories and regions.
- Identify top-performing customers and products.
- Understand profit distribution across product categories.
- Analyze shipping costs across different markets.
- Practice data modeling by merging multiple datasets using Pandas.
- Create visualizations to communicate business insights.

---

## 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook / VS Code
- OpenPyXL

---

## 📂 Dataset

The dataset is divided into four Excel files:

| File | Description |
|------|-------------|
| Sales.xlsx | Sales transactions including Sales, Quantity, and Profit |
| Orders.xlsx | Order information including Customer Name and Order Date |
| Products.xlsx | Product details including Category and Sub-Category |
| Shipping.xlsx | Shipping information including Market, Region, and Shipping Cost |

---

## 📊 Business Questions Solved

### 1. Sales by Category
- Merged **Sales** and **Products**
- Identified the category with the highest sales
- Visualization: Bar Chart

### 2. Sales by Region
- Merged **Sales** and **Shipping**
- Compared sales across different regions
- Visualization: Bar Chart

### 3. Sales by Customer
- Merged **Sales** and **Orders**
- Identified the top customers by total sales
- Visualization: Bar Chart

### 4. Profit by Category
- Merged **Sales** and **Products**
- Compared profit generated by each category
- Visualization: Pie Chart

### 5. Shipping Cost by Market
- Used **Shipping** table
- Compared shipping costs across different markets
- Visualization: Bar Chart

### 6. Top 10 Products by Sales
- Used **Sales** table
- Ranked products based on total sales
- Visualization: Bar Chart
